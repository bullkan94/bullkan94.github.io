# 느낌
## 달리기
템포런을 뛰면서 하프 마라톤을 뛸때의 감각을 다시 찾은 것 같다. 대회 이후 훈련에서 오른쪽 종아리 통증을 느끼고 자세가 잘못 되었다고 생각해서 바꾸고 있었는데, 자세가 잘못된게 아니고 회복이 안돼서 아팠던게 아닌가 싶다.  
한동안 이 자세를 유지해보면서 내 생각이 옳은지 확인해봐야 할 것 같다.  

## 케틀벨
오른쪽과 왼쪽이 각각 특징이 있는 것 같다. 한발 데드리프트는 왼쪽발을 디디고 있을때 가장 잘 된다. 반대로 불가리안 스쿼트는 오른발을 디디고 있을때 잘 된다.  
그리고 오른발은 데드리프트를 할때 뭔가 중심이 잘 안맞는듯한 느낌이 든다. 오른발이 틀어진걸까? 까먹지 않는다면 여러 방향에서 촬영하면서 자세를 보면 좋을 것 같다.  

## 근력운동
이번주는 아주 운동이 힘들게 느껴졌다. 물론 저번주도 힘들긴 했지만 뭔가 정신적으로 더 피곤하다고 느끼는 것 같다.  
점진적 과부화는 순조롭게 진행되고 있어서 다음 운동 사이클에는 어느정도 목표에 도달할 것 같다. 슬슬 다음 운동 강도로 넘어가기 위해 어떤 운동으로 대체하거나 프로그램을 짤지 고민하면 좋을 것 같다.  

## 전체 느낌
근력운동과 케틀벨을 하기 전에 별로 의욕이 안나거나 굉장히 힘들게 느껴졌다. 알게모르게 신경계가 피로해지고 있는걸까? 다음주가 디로딩 기간인데, 디로딩 후에 어떤 느낌이 들지 기대된다.  
달리기 강도를 어떻게 조절해나갈지 조금 고민된다. 거리를 늘리면 운동 시간이 길어져서 고민이고, 속도를 높이게 되면 심박수를 넘기게 된다... 중간 중간 거리를 깎고 속도를 높이는 식으로 강도를 조정해야할까? 적절한 스윗 스팟이 있을지 여러모로 시도해봐야겠다.
