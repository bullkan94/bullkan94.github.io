# 느낌
## 조깅
14km 를 뛰는 것은 처음이기 때문에 혹시나 싶어서 조깅 강도를 크게 낮춰서 약 40분 하는걸로 바꿨다. 거리주를 뛰고 나니 꽤 좋은 결정이었던 것 같다.  

## 거리주
8km 지점에서 에너지젤 보급을 연습해보았다. 생각보다 다 먹는데 시간이 걸리기 때문에 (대략 2~300m?) 급수 전에 미리 9km 지점쯤에서 먹어두는게 좋을 수도 있을 것 같다.  
8km 까지는 크게 힘들지 않았지만 10km 를 넘고 나서는 다리가 매우 무겁게 느껴졌다. 그런데 시계를 보면 꽤 목표 페이스를 유지하는 편이었다. 아마 초반부에서 페이스 조절을 잘못하여 빠르게 달린 영향도 있겠지만, 적당히 다리에 힘을 빼고 뛰는 느낌을 잡은 것일 수도 있을 것 같다.  


## 훈련 스케줄
앞으로 17km주가 남아있다. 근데 후반에 다리가 잠기는 느낌이 들었기 때문에 16km 만 뛰어보는게 보수적으로 접근하는 관점에서 좋을 것 같다.  
16km주도 잘 끝낼 수 있을지 걱정 되지만, 적어도 회복에 일주일이 소요 되지는 않을 것 같아서 너무 걱정하지는 않아도 될 것 같다. 모의 레이스라고 생각하기보다는 훈련 중의 하나로 생각하고 접근하는게 좋을 것 같다.
