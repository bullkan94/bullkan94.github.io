# 느낌
## 달리기
뭔가 점점 달리기 할때 몸이 무거워진다는 느낌이 든다. 정확히는 다리가 뭔가 더 힘든 것 같은데... 내 달리는 자세나 훈련 스케줄이 적당하지 않은걸까? 아니면 신발이 오래돼서 바꿀때가 된걸까? 슬슬 신발을 바꿀 타이밍이 오긴해서 적당한 타이밍에 하나 사도록 해야겠다.  
그리고 이번주는 상당히 추워서 뛰려는 의지가 곧잘 꺾였다... 다음주는 좀 덜 추웠으면 좋겠다.  

## 케틀벨
무게를 16kg로 올렸다. 생각보다 할만하긴 했는데 확실히 저항이 올라가서 그런지 심박이 많이 올라간다.  
상체는 근력운동으로 커버하고 있고 다리 운동이 너무 없는 것 같아서 클린 대신 불가리안 스쿼트로 바꿨다. 익숙하지 않은 자세이고 연습이 안돼서 그런지 상당히 힘들게 느껴진다. 5개도 충분히 힘들긴 했는데 아마 자세가 불편해서 그런게 더 클 것 같다. 일단 익숙해지는데 집중하자.  

## 근력운동
정말 신기한건 주초에 횟수를 1씩 (혹은 3~5초) 올리면 그렇게 힘들게 느껴진게, 주말이 오면 힘들긴해도 주초보단 할만해진다는 것이다. 몸이 적응하면서 강해진다는 느낌이 들어서 좀 재밌어지는 것 같다.  

## 전체 느낌
저번주부터 16:8 간헐적 단식으로 체중관리를 시도하고 있다. 아마 수분이 빠진거겠지만 그래도 체중 변화가 보여서 기분이 좋긴하다.  
달리기는 뭔가 발전이 보이지 않고 퇴화하는 것 같은 느낌이 든다... 뭐가 문제인걸까? 신발도 바꿔보고 훈련 스케줄도 바꿔보면서 조사를 해봐야 할 것 같다.  
생각해보면 25년은 거의 맨몸운동을 안하고 달리기만 했던 것 같다. 몸 건강에도 썩 좋지 않고 체중관리나 체형관리 측면에서도 근력운동을 넣는게 중요하니까 꼭 26년도에는 이 운동 습관을 쭉 유지할 수 있었으면 좋겠다.
